# Jogo de Corrida com Combate e Upgrades

Este é um jogo de corrida top-down desenvolvido para rodar diretamente no GitHub Pages. 

## Funcionalidades
- **Corridas de 3 Voltas**: Dispute contra IAs em circuitos dinâmicos.
- **Combate com Armas**: Use seu arsenal para tirar os oponentes da pista.
- **Circuitos Aleatórios**: Cada corrida oferece um traçado novo e desafiador.
- **Garagem & Upgrades**: Use o dinheiro das vitórias para melhorar motor, blindagem e armas do seu carro.
- **Customização**: Altere a aparência do seu veículo.

## Como Jogar
1. Clone o repositório ou acesse via GitHub Pages.
2. Use as setas do teclado ou WASD para pilotar.
3. Use a barra de espaço para atirar.
4. Ganhe corridas para acumular capital e investir na garagem.

## Desenvolvimento
Desenvolvido por Antigravity (IA) conforme solicitação do usuário.
Para deploy no GitHub Pages:
- Certifique-se de que o `index.html` está na raiz.
- Ative o GitHub Pages nas configurações do repositório apontando para a branch `main`.
